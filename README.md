# p-lod-data

Downloadable data in various formats for the Pompeii Linked Open Data initiative.

Everything here is in its earliest stages. The prefixes don't dereference right now and will almost certainly change. Also, the data here reflects what has been loaded into Omeka S. There is more coming...

Read more about the project at https://palp.p-lod.umasscreate.net/ .
